# Existing server landscape

## Main projects reviewed
- AminForou `mcp-gsc` — local/community GSC wrapper with many tools
- AminForou `google-search-console-mcp-v2` — remote/hosted concept
- Shin-sibainu `google-search-console-mcp-server` — local OAuth-driven package for Claude Code / Cursor
- surendranb `google-search-console-mcp` — Python + service account oriented
- saurabhsharma2u `search-console-mcp` — broader Search/Bing/GA4 platform

## Useful pattern categories
- **small wrapper**
- **local OAuth UX**
- **service-account automation**
- **remote hosted auth portal**
- **multi-platform SEO suite**

## What this project should learn
- keep GSC semantics strong
- avoid becoming too broad too early
- treat auth and security as first-class design concerns
- do not hide Search Console caveats
