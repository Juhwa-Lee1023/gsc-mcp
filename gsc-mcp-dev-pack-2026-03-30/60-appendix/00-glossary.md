# Glossary

## GSC terms
- **Property** — a Search Console unit representing a site or URL branch.
- **Domain property** — covers all protocols and subdomains for a domain.
- **URL-prefix property** — covers only the specified prefix including protocol.
- **Search Analytics** — the Search Console API area for performance data.
- **Search appearance** — feature-specific result class such as AMP or rich results.
- **URL Inspection** — inspection of Google’s indexed view of a URL.
- **Anonymized query** — a privacy-protected query omitted from row-level output.
- **Bulk export** — daily Search Console export to BigQuery.

## MCP terms
- **Host** — the AI application.
- **Client** — the connector inside the host.
- **Server** — the MCP-capable external service.
- **Tool** — an invocable operation.
- **Resource** — contextual data addressed by URI.
- **Prompt** — a parameterized prompt template exposed by a server.
- **STDIO** — local subprocess transport.
- **Streamable HTTP** — modern HTTP transport for remote MCP.
