# Open questions

## V1-blocking? Usually no.
- Which Node package manager to standardize on?
- Which keychain library to use across macOS / Windows / Linux?
- Which local cache backend to prefer if SQLite packaging becomes annoying?
- How much of the sync worker to scaffold in v1?
- Whether prompt templates should live in code, Markdown, or both?
- Whether opaque cursors should be introduced immediately or after a first raw-offset iteration?

## V2-oriented
- When to add remote Streamable HTTP?
- Whether to support multiple linked Google accounts in the same local install?
- Whether to add BigQuery bulk export routing natively or through a plugin?
- Whether to provide a separate admin UI for allowlist and policy editing?

## Hard no unless separately justified
- generic indexing tool
- kitchen-sink SEO analytics scope
- background crawling features that blur product boundaries
